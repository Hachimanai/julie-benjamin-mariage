# Specification: Finalize RSVP and Visual Polishing

## Goal
Finalize the RSVP functionality to ensure a smooth guest experience and apply visual refinements based on the client's provided feedback images.

## Functional Requirements
- **RSVP Logic:** Guests can select "Confirm" or "Decline" for the main event and potentially related events (e.g., brunch).
- **Interactive Feedback:** Clear visual state changes when an option is selected.
- **Validation:** Ensure required fields (names) are filled before submission.
- **Client Feedback:** Address specific visual adjustments identified in `retours_client/` images.

## Technical Constraints
- **Vanilla JS:** Use pure JavaScript for all logic.
- **Responsive:** Must work perfectly on mobile devices.
- **Design:** Adhere strictly to the Revelia Studio aesthetic (colors, typography).
