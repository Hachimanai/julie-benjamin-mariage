# Tech Stack

## Core Technologies
- **HTML5:** Semantic structure for the invitation content.
- **CSS3:** Custom styling using modern features (Flexbox/Grid), adhering to the Revelia Studio aesthetic.
- **JavaScript (ES6+):** Pure JS for interactivity (countdown, RSVP logic) and DOM manipulation.

## External Resources
- **Google Fonts:** Usage of "Inter", "Cormorant Garamond", and "Great Vibes".

## Development Principles
- **No Frameworks:** Direct use of browser APIs to keep the codebase lightweight and maintainable.
- **Responsive Design:** Mobile-first approach with a focus on elegant presentation across devices.
- **Code Quality:** Clean, idiomatic code without external runtime dependencies.
