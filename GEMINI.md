# Directives du Projet - Invitation Julie & Benjamin

## Vision et Référence
- **Modèle de Structure** : L'architecture visuelle (disposition, espacements, types de blocs) est calquée exclusivement sur le fichier `invite-reveliastudio-invitation-sarah-thomas-copy-1771947291770-2026-04-18-15_12_33.png`.
- **Esthétique** : Style "Revelia Studio" - Luxueux, épuré, avec une colonne centrale blanche se détachant d'un fond crème.

## Palette de Couleurs & Typographie
- **Couleurs Principales** :
  - **Bleu Roi** (`#4169E1`) : Utilisé pour tous les accents, icônes, boutons (état actif) et titres mis en valeur.
  - **Beige Parchemin Clair** (`#faf7f2`) : Utilisé pour les fonds des sous-sections internes.
  - **Fond de Page** (`#f2eee9`) : Teinte neutre pour l'arrière-plan global (body).
- **Typographie** :
  - **Inter** : Police standard pour le corps de texte, les labels et les boutons.
  - **Cormorant Garamond** : Police Serif pour les titres de section et les dates.
  - **Great Vibes** : Police cursive élégante pour les noms des mariés.

## Données du Mariage
- **Mariés** : Julie & Benjamin.
- **Date** : Samedi 24 juillet 2027.
- **Lieu** : Domaine de la Rainville, Villampuy.

## Stack Technique
- **Framework** : Aucun (Vanilla HTML5 / CSS3 / JavaScript ES6).
- **Maintenance** : Code minimaliste, sans dépendances externes (sauf Google Fonts).
- **Layout** : Responsive (mobile-first), colonne centrale de `500px` max, espacement inter-section de `80px` (via padding vertical de `40px`).

## Règles de Modification
- **Alignement** : Les icônes de section (`.icon-header`) doivent toujours être alignées horizontalement avec leur titre.
- **Bords** : Toutes les sous-sections (`.inner-beige`) doivent avoir un `border-radius: 1rem`.
- **Interactivité** : Utiliser JavaScript pur pour le compte à rebours et la gestion des boutons de présence (pas d'inputs radio visibles).
